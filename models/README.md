# Plant Disease Classification

## Overview
Classifying plant leaf diseases from images using pre-trained deep learning models.  
Goal: Automatically detect plant diseases to help farmers and researchers.

## Models
- MobileNetV2  
- EfficientNetB3  
- ResNet50

## Tools
Python, TensorFlow/Keras, NumPy, Matplotlib, Google Colab, GitHub

## How to Run
1. Open a notebook from `models/` in Colab.  
2. Upload dataset if needed.  
3. Run cells sequentially to train and evaluate models.
